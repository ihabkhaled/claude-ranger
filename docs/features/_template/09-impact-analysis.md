# 09 - Full Impact Analysis

## Purpose

Reveal the full blast radius so no team or system is surprised later.

## Step-by-Step Workflow

1. Review impact across technical, operational, and support functions.
2. Document compatibility, migration, monitoring, support, training, and compliance implications.
3. Identify owners and mitigation for each affected area.
