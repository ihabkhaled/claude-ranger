# 19 - Security Review

## Purpose

Record security validation, controls, findings, and approval status.

## Review Checklist

### Identity and Access

- [ ] Authentication path reviewed
- [ ] Authorization path reviewed
- [ ] Permission denial cases tested
- [ ] Tenant isolation reviewed

### Data and Privacy

- [ ] Sensitive data classification reviewed
- [ ] Logging leakage reviewed
- [ ] Retention and deletion impact reviewed
- [ ] Encryption or transport protection reviewed

### Application Security

- [ ] Input validation reviewed
- [ ] Output handling reviewed
- [ ] Abuse controls reviewed
- [ ] File handling reviewed
- [ ] Dependency risk reviewed

### Operational Security

- [ ] Secrets handling reviewed
- [ ] CI/CD risk reviewed
- [ ] Deployment safety reviewed
- [ ] Audit logging reviewed

Do not close this phase if:

- critical or high-risk issues remain unresolved without waiver
- review scope is unclear
- conditions of approval are not explicit
- remediation ownership is missing
